Sortable

interface Sortable

Contract describing that an object can be sorted

The key words “MUST”, “MUST NOT”, “REQUIRED”, “SHALL”, “SHALL NOT”, “SHOULD”, “SHOULD NOT”, “RECOMMENDED”, “MAY”, and “OPTIONAL” in the comments are to be interpreted as described in RFC 2119 {@link http://tools.ietf.org/html/rfc2119}

Method Details

sort()

Sort an object

Takes an OPTIONAL bitwise flags that defines the sort order and sort type. Some flags can be combined to cutomise how the sorting is performed. This function sorts objects in-situ, modifying the original object. It SHOULD NOT produce a return value.
