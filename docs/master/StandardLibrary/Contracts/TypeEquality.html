bool equals(Type $object)

Compare two {@link Type} objects

Takes an {@link Type} object as an argument and compares it with itself ($this) to determine equality between the two objects. Should return boolean true if they are equal, false otherwise. The method may throw an exception if an error occurrs when trying to compare the objects.
